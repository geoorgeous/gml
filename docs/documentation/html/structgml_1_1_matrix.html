Detailed Description

template<typename T, unsigned int Major, unsigned int Minor = Major>
struct gml::Matrix< T, Major, Minor >

A struct templates for matrices of floating-point values.

This struct is used to represent two-dimensional matrices of various sizes.

Note
All GML matrix data is stored in column-major order, i.e.: the contiguous memory of a matrix is stored column after column. You may often need a pointer to this data, the best way to get this is to simply use &someMatrix[0][0];. This provides a pointer to the matrix's first element.
